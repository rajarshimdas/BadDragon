Single File PHP ORM

Powerful lightweight ORM implemented in a single PHP file. Supports MySQL, PostgreSQL and SQLite.

1. Installation

Copy the ORM file into your project.

project/
 ├─ orm.php
 ├─ index.php
 └─ howtouse.html

2. Connect to Database

ORM is loaded by BadDragon

            DB::connect([
                "driver"    => "mysql",
                "host"      => "localhost",
                "database"  => "mydb",
                "username"  => "root",
                "password"  => "password",
                "port"      => "port"       # Optional
            ]);

Supported drivers:

Driver Description
mysql MySQL / MariaDB
pgsql PostgreSQL
sqlite SQLite database

3. Creating Models

Create models that extend the base Model class.

class User extends Model
{
    protected static $table = "users";
}

class Post extends Model
{
    protected static $table = "posts";
}

4. Insert Records

$user = new User();

$user->name  = "John";
$user->email = "john@example.com";

$user->save();

5. Fetch Records

Get all rows

$users = User::all();

Find by ID

$user = User::find(1);

6. Query Builder

Basic where query

$users = User::query()
        ->where("id", ">", 10)
        ->get();

Ordering

$users = User::query()
        ->orderBy("id","DESC")
        ->get();

Limit

$users = User::query()
        ->limit(10)
        ->get();

7. Update Record

$user = User::find(1);

$user->name = "Updated Name";

$user->save();

8. Delete Record

$user = User::find(1);

$user->delete();

9. Relationships

User has many Posts

class User extends Model
{
    protected static $table = "users";

    public function posts()
    {
        return $this->hasMany(Post::class, "user_id");
    }
}

Post belongs to User

class Post extends Model
{
    protected static $table = "posts";

    public function user()
    {
        return $this->belongsTo(User::class, "user_id");
    }
}

Usage

$user = User::find(1);

$posts = $user->posts();

10. Transactions

DB::begin();

try {

    $user = new User();
    $user->name="Transaction Test";
    $user->save();

    DB::commit();

}
catch(Exception $e){

    DB::rollback();

}

11. Raw SQL

$stmt = DB::query(
    "SELECT * FROM users WHERE id=?",
    [1]
);

$row = $stmt->fetch();

12. Pagination Example

$users = User::query()
            ->limit(10)
            ->offset(20)
            ->get();
