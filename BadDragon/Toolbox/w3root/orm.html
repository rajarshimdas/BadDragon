Ultra ORM Documentation

Ultra ORM is a lightweight, single-file PHP ORM designed for high performance and minimal dependencies.

Supported databases:

  • MySQL
  • PostgreSQL
  • SQLite

1. Installation

Download the file:

ultraorm.php

Include it in your PHP project:

require "ultraorm.php";

2. Database Connection

DB::connect([
    'driver'   => 'mysql',
    'host'     => 'localhost',
    'database' => 'myapp',
    'username' => 'root',
    'password' => 'secret',
    'port'     => 3306
]);

3. Creating Models

class User extends Model
{
    protected static $table = "users";
}

4. Insert Record

User::create([
    "name"  => "John",
    "email" => "john@example.com"
]);

5. Find Record

$user = User::find(10);

6. Get All Records

$users = User::all();

7. Update Record

User::updateById(5,[
    "name" => "Updated Name"
]);

8. Delete Record

User::deleteById(5);

9. Query Builder

$users = Query::table("users")
            ->where("age",">",18)
            ->orderBy("name")
            ->limit(10)
            ->get();

10. Get First Record

$user = Query::table("users")
            ->where("email","=","john@mail.com")
            ->first();

11. Bulk Insert

Query::table("logs")->bulkInsert([
    ["event"=>"login","user_id"=>1],
    ["event"=>"logout","user_id"=>1]
]);

12. Pagination

$posts = Query::table("posts")
            ->paginate(2,20);

Page 2 with 20 records per page.

13. Relationships

hasMany

$posts = Model::hasMany(Post::class,"user_id","id",1);

belongsTo

$user = Model::belongsTo(User::class,"user_id",$post);

14. Transactions

DB::begin();

try{

    User::create([
        "name"=>"Alice"
    ]);

    User::create([
        "name"=>"Bob"
    ]);

    DB::commit();

}catch(Exception $e){

    DB::rollback();

}

15. Migrations

Migration::create("users","
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100),
    email VARCHAR(200),
    created_at DATETIME,
    updated_at DATETIME
");

16. Query Profiling

print_r(DB::profile());

Shows all executed queries and execution time.

17. Security

  • Prepared statements for values
  • Identifier validation
  • Operator whitelist
  • Limit/offset casting
  • PDO emulation disabled

18. Performance Tips

  • Use bulkInsert for large data
  • Use limit when fetching records
  • Use paginate for large tables
  • Avoid SELECT *
